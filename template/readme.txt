A term PSMS means that  Pixel Shift Multi Shooting. If the number after PSMS in the folder name was 16; one tiff was developed for every 16 raw images in this folder. If the number after PSMS in the folder name was 4; one tiff was developed for every 4 raw images in this folder. If the number after PSMS in the folder name was 0; Pixel Shift Multi Shooting was not conducted, and every raw images should be developed individually.
